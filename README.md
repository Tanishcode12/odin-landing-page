# Odin Landing Page Practice
This project is a practice exercise in HTML and CSS for building a modern landing page. It demonstrates layout design, typography, responsive elements, and styling with CSS.
# Project Overview
The landing page includes:
1. Header Section
> - Logo and navigation links.
> - Horizontal layout using Flexbox.
2. Hero Section
- Main headline and subtext.
- Call-to-action button (Sign Up).
- Hero image.
3. Information Section
> - Grid-like layout with multiple cards.
- Each card contains an image and text.
> - Quote / Testimonial Section
> - Displaying a centered quote with author attribution.
4. Call-to-Action Section
> - Background-colored section with another Sign Up button.
5. Footer Section
> - Simple copyright notice.
# Features
- Fully structured HTML5 document.
- Styled with CSS Flexbox and Google Fonts.
- Responsive layout basics (flex-based).
- Buttons, links, and images included.
- Multiple sections with visually distinct backgrounds.
